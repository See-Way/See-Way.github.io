Windows运行多个版本Nodejs

发表于2020-11-21|更新于2020-11-21|环境配置
|阅读量:

nvm可以很好地解决Windows环境下Nodejs版本切换问题

安装

  • 卸载Nodejs

  • 删除...\nodejs和...\npm文件夹

  • 从Github上下载nvm

  • [配置镜像]找到nvm所在文件夹，打开settings.txt，在最后加入两行代码

使用

  • nvm install x.x.x：安装x.x.x版本Nodejs
  • nvm ls：查看所有已安装Nodejs的版本
  • nvm use x.x.x：使用x.x.x版本的Nodejs
